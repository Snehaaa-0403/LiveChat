## 📌 Project Overview

**LiveChat** is a high-performance messaging application built with the MERN stack. It features a modern, responsive UI and a robust real-time engine, allowing users to connect, share media, and communicate instantly across a secure platform.

## ✨ Key Technical Features

* **Instant Messaging & Real-Time Presence:** Powered by **Socket.io**, the app supports seamless, low-latency text communication. It features a live **Global Presence Tracker** that broadcasts "Online/Offline" status updates across the community, ensuring users always know who is available to chat.
* **Rich Media Integration & Profile Customization:** Integrated with **Cloudinary**, users can exchange high-quality images within conversations and personalize their identity with custom profile avatars. The system handles asynchronous image uploads and cloud storage efficiently to ensure fast message delivery.
* **Reactive Global State with Zustand:** Implemented **Zustand** for centralized, high-speed state management. This handles everything from real-time message streams and active user lists to authentication persistence, ensuring the UI stays in sync with the database without unnecessary page reloads.

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite), Tailwind CSS, **Zustand**, Lucide React
* **Backend:** Node.js, Express.js
* **Real-Time:** **Socket.io** (WebSockets)
* **Database:** MongoDB Atlas (Mongoose)
* **Storage:** Cloudinary API
* **Deployment:** Render (Static Site & Web Service)
